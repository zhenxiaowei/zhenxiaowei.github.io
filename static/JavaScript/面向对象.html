面向对象

一、概念

对象：任何东西都是对象（属性和方法的集合）
面向对象：和对象打交道

1、对象的组成
属性：有归属的变量
方法：有归属的函数
事件：在js面向对象里，视同为方法

2、类、实例
类：模板，没有实际的功能
实例：根据模板（类）造出的一个具体的东西
function Person(){}---Person类
let p1=new Person()---p1实例
类--实例化--实例

3.面向对象的三大特点：
1、封装：把细节包起来，用户只管只有，不要关心是怎么实现的
2、继承：子类可以直接使用父类的功能（东西）
3、多态：在js中没有
能new出来的，肯定是函数
任何东西都是对象
new一个类的实例，这个类的构造函数自动执行

二、创建对象

1、字面量方式创建对象，并访问属性和方法

(1)字面量创建对象，空对象

var obj={};

(2)字面量创建对象，非空对象

var obj={ name:"zxw", age:30, run:function(){ console.log("会跑") }, sleep:function(hours){ console.log("睡"：+hours+"个小时") } }

(3)访问对象的属性和方法

console.log(obj["name"]) console.log(obj.name) obj["run"](); obj.sleep(10);

(4)动态给对象添加属性
注：后期添加的属性，在这之前使用该对象，也会有当前添加的属性

obj.faceValue=100 ;

(5)给对象添加方法

obj.play=function(){ console.log("玩") }

(6)动态删除对象属性

delete obj.name console.log(obj.name) //undefined

(7)动态删除对象的方法

delete obj.run obj.run()//报错

2、通过构造函数创建对象

通过该方法创建对象，会自动执行构造函数

function Person(name,age){ this.name=name; this.age=age; this.job=function(gz){ console.log(gz+"前端开发") //函数对象方法 } alert("123")//构造函数 } var child=new Person("Zxw",30); //创建对象时，会执行构造函数一次，弹出123 child.job("我的工作是") //调用对象的方法，会弹出前端开发

3、工厂模式创建对象

在一个函数内创建好对象，把这个返回

function cratePerson(name,age){ var o=new Object() o.name=name; o.age=age; return o; } var person1=cratePerson("Zxw1",1); var person2=cratePerson("Zxw2",2);

4、原型模式创建对象

function Person(){}; Person.prototype.name="zxw"; Person.prototype.age=30; Person.prototype.job=function(){ }; var Person1=new Person(); var Person2=new Person(); console.log(Person1.name) //调用这个对象的共享属性 console.log(Person2.job()); //调用这个对象的共享方法

5、构造函数模式和原型模式组合使用 (混合模式)

构造函数用于定义实例属性，原型模式用于定义方法和共享的属性

function Person(name.age){ this.name=name; this.age=age; } Person.prototype={ constructor:Person, sayName:function(){ alert(this.name) } } var Person1=new Person("zxw",1) Person1.sayName();
