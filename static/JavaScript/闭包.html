闭包

定义

一个函数中有一个内部函数，这个内部函数可以访问外层函数的局部变量、局部方法、参数等， 当其中这样的内部函数在包含它们的外部函数之外被调用，就会形成闭包。

闭包的三个特性

1、函数嵌套函数
2、函数内部可以引用外部的参数和变量
3、参数和变量不会被垃圾回收机制回收

1、第一种写法

//这种方法只是给函数添加了一些属性，形成闭包 function AA(r) { this.r = r; } AA.prototype.AAchild = function () { return this.r; } var c = new AA(1); console.log(c.AAchild());

2、第二种写法

//返回值是一个匿名函数，形成闭包 var BB = function () { var obj = new Object(); obj.name = 'zxw'; obj.getName = function () { return obj.name; }; return obj; }; var b = new BB(); console.log(b.getName());

3、第三种写法

//一个对象中的方法，调用这个对象的一个属性，形成闭包 var CC = new Object(); CC.name = 'zxzz'; CC.CCchild = function () { return this.name; }; console.log(CC.CCchild());

4、第四种写法

//一个对象中的方法，调用这个对象的一个属性，形成闭包 var DD={ 'name':'ggg', 'ddChild':function () { return this.name } }; console.log(DD.ddChild());
